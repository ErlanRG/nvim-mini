# nvim-mini

Minimal Neovim configuration.

## Plugins

- [lazy.nvim](https://github.com/folke/lazy.nvim) — plugin manager
- [formatter](https://github.com/stevearc/conform.nvim) — file explorer
- [mason.nvim](https://github.com/mason-org/mason.nvim) — LSP server installer
- [nvim-cmp](https://github.com/hrsh7th/nvim-cmp) — autocompletion
- [nvim-lspconfig](https://github.com/neovim/nvim-lspconfig) — LSP client configs
- [nvim-treesitter](https://github.com/nvim-treesitter/nvim-treesitter) — treesitter
- [oil.nvim](https://github.com/stevearc/oil.nvim) — file explorer
- [snacks.nvim](https://github.com/folke/snacks.nvim) — QoL plugins
- [mini.nvim](https://github.com/nvim-mini/mini.nvim) — QoL plugins
